Les bases de Git

git init

Cette commande permet d'initialiser git sur le projet voulu

git status

Nous donne le status des fichiers:

  • untracked (non suivi): le fichier n'est pas suivi par git
  • added (ajouté): le fichier vient d'etre ajouté à la zone de suivi
  • modifié (not staged): le fichier est modifié, mais ces modifications ne feront pas partie de la prochaine sauvegarde

git add monFichier

Cette commande ajoute monFichier à la zone de suivi

git commit -m "commentaire à ajouter"

Cette commande permet de faire une sauvegarde de notre travail à cet instant précis. Ne sera sauvergardé que ce qui est présent dans la zone de suivi

git remote add origin url http://monurl.com

Cette commande permet d'ajouter notre url à git(http://monurl.com). Cette url aura pour surnom origin

git branch -M main

Cette commande permet de renommer notre branche courante "master" en "main"

git push -u origin main

Cette commande permet d'envoyer le contenu de notre repertoire en ligne sur l'url conservé dans "origin". On a précisé qu'on a envoyé (push) dans notre branche "main".
